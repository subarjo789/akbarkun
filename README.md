# 📚 Web Scraping Books

Project sederhana untuk scraping data buku dari [books.toscrape.com](http://books.toscrape.com/) menggunakan Python.

## 🎯 Fitur
- Scraping judul buku, harga, rating, dan stok
- Menyimpan data ke format JSON dan CSV
- Delay antar request agar tidak membebani server

git clone https://github.com/username/web-scraping-project.git
cd web-scraping-project
